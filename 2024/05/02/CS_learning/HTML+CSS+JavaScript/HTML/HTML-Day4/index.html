头部元素

Title

<title>
定义文档的标题，显示在浏览器的标题栏或页面标签上

base

<base>

  1. 为页面上的所有链接规定默认地址或默认目标
  2. 只能在<head>元素中使用
  3. target属性规定在何处打开链接(默认是在当前窗口中打开，_blank在新窗口中打开，_self在相同的框架中打开，_parent在父框架中打开，_top在整个窗口中打开，_framename在指定的框架中打开，如果没有找到该框架，则在新窗口中打开)

link

<link>
定义文档与外部资源的关系，通常用于链接样式表，也可以用于定义网站图标。

这段代码的意思是，将styles.css文件应用到当前文档中，rel属性定义文档与被链接文档之间的关系，type属性定义被链接文档的MIME类型，href属性定义被链接文档的位置，href属性的值可以是绝对URL或相对URL，如果是相对URL，那么相对的是包含<link>元素的文档。

meta

<meta>
提供有关页面的元信息，比如描述、关键词、作者等

<meta>标签提供了元数据.元数据也不显示在页面上，但会被浏览器解析。

META 元素通常用于指定网页的描述，关键词，文件的最后修改时间，作者，和其他元数据。

元数据可以使用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他Web服务。

style

<style>
定义文档的样式信息

HTML CSS

CSS是什么

CSS 指层叠样式表（Cascading Style Sheets），用来描述网页的样式，包括布局、颜色和字体等

CSS 是一种样式表语言，用来描述 HTML 元素应该如何显示在屏幕、纸张或其他媒体上。

CSS 可以通过以下方式添加到HTML中:

  • 内联样式- 在HTML元素中使用”style” 属性
  • 内部样式表 -在HTML文档头部 <head> 区域使用<style> 元素 来包含CSS
  • 外部引用 - 使用外部 CSS 文件

CSS 语法

CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明

  • 选择器（selector）: 选择器是您需要改变样式的 HTML 元素
  • 属性（property）: 属性是您要修改的样式属性（比如字体、颜色、边框等）
  • 值（value）: 值是您要给属性设置的值

CSS 注释

CSS 注释以 /* 开始，以 */ 结束

内部样式表

当单个文档需要特殊的样式时，就可以使用内部样式表。你可以在<head>部分通过<style>标签定义内部样式表

外部样式表

当样式需要应用到很多页面时，外部样式表将是理想的选择。使用外部样式表，你就可以通过更改一个文件来改变整个站点的外观。

外部样式表通常存储在 CSS 文件中。然后，多个页面可以引用同一个样式表。

CSS 文本

CSS 文本属性允许您控制文本的外观

  • text-align 属性用于设置文本的水平对齐方式
  • text-decoration 属性用于设置文本的装饰效果

HTML表格基础

HTML 表格由 <table> 元素来定义。表格中的每一行由 <tr> 元素定义。表格数据由 <td> 元素定义。表头由 <th> 元素定义。

<table> 元素表示整个表格，它包含两个主要部分：<thead> 和 <tbody>。

<thead>用于定义表格的标题部分: 在<thead>中，使用<th>元素定义列的标题，以上实例中列标题分别为”列标题1”，”列标题2”和”列标题3”。

<tbody> 用于定义表格的主体部分: 在 <tbody> 中，使用 <tr> 元素定义行，并在每行中使用 <td> 元素定义单元格数据，以上实例中有两行数据，每行包含三个单元格。

通过使用 <th> 元素定义列标题，可以使其在表格中以粗体显示，与普通单元格区分开来。

HTML 表格还可以具有其他部分，如 <tfoot> （表格页脚）和 <caption> （表格标题），<tfoot> 可用于在表格的底部定义摘要、统计信息等内容。 <caption> 可用于为整个表格定义标题。

HTML 表格还支持合并单元格和跨行/跨列的操作，以及其他样式和属性的应用，以满足各种需求。

我们也可以使用 CSS 来进一步自定义表格的样式和外观。

HTML区块

HTML 可以通过 <div> 和 <span>将元素组合起来。

HTML 区块元素

大多数 HTML 元素被定义为块级元素或内联元素。

块级元素在浏览器显示时，通常会以新行来开始（和结束）。

实例: <h1>, <p>, <ul>, <table>。

HTML 内联元素

内联元素在显示时通常不会以新行开始。

实例: <b>, <td>, <a>, <img>。

<div> 元素

<div> 元素是块级元素，它是用于组合其他 HTML 元素的容器。

<div> 元素没有特定的含义。除此之外，由于它属于块级元素，浏览器会在其前后显示一个换行。

如果与 CSS 一同使用，<div> 元素可用于对大的内容块设置样式属性。

<span> 元素

<span> 元素是内联元素，可用作文本的容器，以便通过 CSS 来对文本设置样式。

<span> 元素也没有特定的含义。当与 CSS 一同使用时，<span> 元素可用于为部分文本设置样式属性。

HTML 表单

HTML 表单基础

HTML 表单用于收集用户输入。表单可以包含文本域、复选框、单选框、提交按钮等等。

表单使用表单标签<form>来设置表单，表单中的输入项使用输入标签<input>来设置。

HTML 表单元素

  • <form> 元素用于创建表单，action 属性定义了表单数据提交的目标 URL，method 属性定义了提交数据的 HTTP 方法。
  • <label> 元素用于为表单元素添加标签，提高可访问性。
  • <input> 元素是最常用的表单元素之一，它可以创建文本输入框、密码框、单选按钮、复选框等。type 属性定义了输入框的类型，id 属性用于关联 <label> 元素，name 属性用于标识表单字段。
  • <select> 元素用于创建下拉列表，而 <option> 元素用于定义下拉列表中的选项。

HTML URL

URL 是通过互联网传输数据的最常见的方式。

URL - 统一资源定位器

Web浏览器通过URL从Web服务器请求页面。

当您点击 HTML 页面中的某个链接时，对应的 <a> 标签指向万维网上的一个地址。

一个统一资源定位器(URL) 用于定位万维网上的文档。

一个网页地址实例: http://www.runoob.com/html/html-tutorial.html
语法规则:

说明:

  • scheme - 定义因特网服务的类型。最常见的类型是 http
  • host - 定义域主机（http 的默认主机是 www）
  • domain - 定义因特网域名，比如 runoob.com
  • port - 定义主机上的端口号（http 的默认端口号是 80）
  • path - 定义服务器上的路径（如果省略，则文档必须位于网站的根目录中）。
  • filename - 定义文档/资源的名称

常见的 URL Scheme:

  1. http:超文本传输协议,以 http:// 开头的普通网页。不加密。
  2. https:安全超文本传输协议,安全网页，加密所有信息交换。
  3. ftp:文件传输协议,将文件下载或上传至网站。
  4. file:您计算机上的文件。

URL 编码

URL 只能使用 ASCII 字符集.

来通过因特网进行发送。由于 URL 常常会包含 ASCII 集合之外的字符，URL 必须转换为有效的 ASCII 格式。

URL 编码使用 “%” 其后跟随两位的十六进制数来替换非 ASCII 字符。

URL 不能包含空格。URL 编码通常使用 + 来替换空格。

至于编码的详细教程可以参考这个链接：HTML URL 编码
