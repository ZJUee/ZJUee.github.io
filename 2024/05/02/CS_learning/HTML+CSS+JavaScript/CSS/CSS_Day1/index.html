CSS简介

CSS (Cascading Style Sheets，层叠样式表)，是一种用来为结构化文档（如 HTML 文档或 XML 应用）添加样式（字体、间距和颜色等）的计算机语言，CSS 文件扩展名为 .css。

通过使用 CSS 我们可以大大提升网页开发的工作效率！需要学会如何使用 CSS 同时控制多重网页的样式和布局。最后完成HTML关注文件的内容，CSS关注文件的表现。通过改写CSS文件，可以改变整个网站的外观。

CSS3 现在已被大部分现代浏览器支持，而下一版的 CSS4 仍在开发中。

CSS 语法

CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明。

  • 选择器通常是您需要改变样式的 HTML 元素。
  • 每条声明由一个属性和一个值组成。
  • 属性（property）是您希望设置的样式属性（style attribute）。每个属性有一个值。
  • CSS 声明总是以分号 ; 来终止，声明组以大括号 {} 来定义。

id 和 class 选择器

如果你要在HTML元素中设置CSS样式，你需要在元素中设置”id” 和 “class”选择器。

  • id 选择器：使用 id 属性为 HTML 元素指定唯一的样式。id 选择器在样式表中以 “#” 来定义。
  • class 选择器：使用 class 属性为 HTML 元素指定样式。class 选择器在样式表中以 “.” 来定义。

插入样式表

有三种方式插入样式表：

  • 外部样式表
  • 内部样式表
  • 内联样式

外部样式表

当样式需要应用于很多页面时，外部样式表将是理想的选择。在使用外部样式表的情况下，你可以通过更改一个文件来改变整个站点的外观。

优先级

样式表允许以多种方式规定样式信息。样式可以规定在单个的 HTML 元素中，在 HTML 页的头元素中，或在一个外部的 CSS 文件中。甚至可以在同一个 HTML 文档内部引用多个外部样式表。

一般情况下，优先级如下：

（内联样式）Inline style > （内部样式）Internal style sheet >（外部样式）External style sheet > 浏览器默认样式

CSS 字体

CSS 字体属性用于定义字体、加粗、大小等。

字体系列

字体系列属性用于设置文本的字体系列。

CSS 字体系列

  • Serif 字体 - 这些字体具有衬线，即在字体的末端有额外的装饰。
  • Sans-serif 字体 - 这些字体没有衬线，即在字体的末端没有额外的装饰。
  • Monospace 字体 - 这些字体在每个字符之间有相同的间距。
  • Cursive 字体 - 这些字体模仿手写文字。
  • Fantasy 字体 - 这些字体设计为装饰性文字。
  • 默认字体 - 浏览器会显示默认的字体。

CSS 链接

CSS 链接样式属性用于定义链接的样式，可以用任何CSS属性（如颜色，字体，背景等）。

特别的链接，可以有不同的样式，这取决于他们是什么状态。

这四个链接状态是：

  • a:link - 正常，未访问过的链接
  • a:visited - 用户已访问过的链接
  • a:hover - 当用户鼠标放在链接上时
  • a:active - 链接被点击的那一刻
